1. O que são banco de dados? 
   **R1:** Banco de dados é uma coleção de dados que estão armazenados e organizados.

2. Explique o conceito de generalidade e alto grau de abstração de banco de dados.
   **R2:** Generalidade se refere à capacidade de um banco de dados de armazenar diferentes tipos de dados, enquanto o alto grau de abstração permite que os usuários interajam com os dados sem precisar entender os detalhes técnicos de como eles são armazenados.

3. Comente por que é importante a eficiência dos sistemas atualmente?
   **R3:** Para evitar vazamento de informações, garantir maior segurança de dados, entre outros motivos.

4. Como é possível atingir usos muito mais sofisticados dos recursos de dados?
   **R4:** Usando técnicas avançadas de análise de dados, inteligência artificial, e integração de diferentes fontes de dados.

5. Como eram as primeiras arquiteturas de banco de dados?
   **R5:** As primeiras arquiteturas de banco de dados eram baseadas em arquivos simples e sistemas hierárquicos ou de rede, com estrutura rígida e pouca flexibilidade.

6. O que é um SGBDR?
   **R6:** SGBDR (Sistema de Gerenciamento de Banco de Dados Relacional) é um tipo de sistema de gerenciamento de banco de dados que organiza dados em tabelas relacionadas.

7. O que é um SGBD ou Sistema Gerenciador de Bancos de Dados e qual é a sua melhoria em relação ao armazenamento de dados em arquivos?
   **R7:** Um SGBD é um software que facilita a criação, gerenciamento e manipulação de bancos de dados. Ele melhora o armazenamento de dados em arquivos por proporcionar mais eficiência, integridade e segurança.

8. Quais os benefícios de usar um sistema de banco de dados para uma empresa?
   **R8:** Facilita o acesso e a manutenção de informações, permite manter cópias eletrônicas, entre outros benefícios.

9. Se o Administrador do banco de dados (DBA – Database Administrator) não administrar o banco de dados de forma adequada, o que será comprometido?
   **R9:** Todos os dados contidos podem estar em risco, incluindo informações sigilosas.

10. Algumas empresas ainda usam abordagem de processamento tradicional de arquivos, quais motivos disso ainda acontecer?
    **R10:** Falta de tempo e recursos para treinar colaboradores, ausência de necessidade de mudança se tudo está funcionando bem, liderança não familiarizada com tecnologia, e sensação de insegurança dos dados, entre outros motivos.
